Number Guesser Game

Description:
This is a Python-based number guessing game that runs in Docker.

Running Locally:
1. Navigate to the project folder.
2. Run the script using:
   python NumberGuesser-abdi0289.py

Running in Docker:
1. Pull the image from Docker Hub:
   docker pull isaacabdile/number-guesser
2. Run the container:
   docker run -it isaacabdile/number-guesser
